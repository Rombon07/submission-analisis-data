Ikuti panduan Step-by-Step berikut untuk menjalankan proyek ini di komputer lokal Anda.

## 1. Clone Repository (Download Proyek)
Buka terminal (Git Bash atau VS Code) dan jalankan perintah ini untuk mengunduh proyek:

```bash
git clone [https://github.com/Rombon07/submission-analisis-data.git](https://github.com/Rombon07/submission-analisis-data.git)
cd submission-analisis-data
```

## 2. Setup Environment
Pilih salah satu cara di bawah ini untuk menyiapkan lingkungan Python:

Opsi A: Menggunakan Anaconda (Disarankan)
```bash
conda create --name main-ds python=3.9 -y
conda activate main-ds
```
Opsi b: Menggunakan Python venv (Terminal Biasa)
```bash
# Untuk Windows:
python -m venv venv
.\venv\Scripts\activate

# Untuk Mac/Linux:
python3 -m venv venv
source venv/bin/activate
```

## 3. Install Library
Install semua library yang dibutuhkan dengan perintah berikut:
```bash
pip install -r requirements.txt
```

## 4. Jalankan Aplikasi (Run App)
Jalankan perintah ini untuk membuka dashboard di browser:
```bash
streamlit run dashboard/dashboard.py
```
